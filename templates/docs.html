Getting Started with OwO-Dusk

Welcome to the OwO-Dusk documentation. This guide will help you set up and use OwO-Dusk effectively.

What is OwO-Dusk?

OwO-Dusk is an automation tool for the Discord OwO bot. It helps automate repetitive tasks and provides a dashboard to monitor your progress.

ℹ️
Important: This documentation is a work in progress. Some sections may be incomplete.

Requirements

  • Python 3.8 or higher
  • Discord account with access to OwO bot
  • Discord token (for self-bot functionality)

Installation

Follow these steps to install OwO-Dusk on your system:

Clone the Repository

git clone https://github.com/echoqueill/owo-dusk.git
cd owo-dusk

Install Dependencies

pip install -r requirements.txt

Configure Settings

Create a copy of the config.example.json file and name it config.json. Edit the file to add your Discord token and other settings.

Run the Bot

python uwu.py
